Quick mount overview

Rychlý přehled připojení

Data hub (all storage paths):
\\{{ server_name }}\data

Datové centrum (všechny cesty):
\\{{ server_name }}\data

Safe mirrored storage (1:1 mirrored HDDs):
\\{{ server_name }}\data\safe

Bezpečné zrcadlené úložiště (1:1 HDD):
\\{{ server_name }}\data\safe

Fast storage (NVMe, not safe):
\\{{ server_name }}\data\fast

Rychlé úložiště (NVMe, bez ochrany):
\\{{ server_name }}\data\fast

Guest area (write allowed for guests):
\\{{ server_name }}\data\guest

Guest zóna (hosté mohou zapisovat):
\\{{ server_name }}\data\guest

Guest access (read everywhere, write only in \data\guest):
user: nasguest
password: nasguest

Guest přístup (čtení všude, zápis jen v \data\guest):
uživatel: nasguest
heslo: nasguest

User nasuser has extended access; for the password, contact the administrator.

Uživatel nasuser má rozšířený přístup; pro heslo kontaktujte administrátora.

How to connect from Windows: Guide: connect to SMB share on Windows

Jak se připojit z Windows: Návod: připojení SMB sdílení ve Windows

NFS access is also enabled, but is experimental and intended only for advanced users. Character encoding and filename handling may be incorrect when using NFS. For NFS mounting instructions on Windows, see for example this NFS client guide for Windows .

NFS přístup je také povolen, ale je experimentální a určený pouze pro pokročilé uživatele. Při použití NFS může docházet k chybné znakové sadě a problémům s názvy souborů. Návod na připojení NFS ve Windows najdete například v tomto článku o NFS klientu pro Windows .

Windows: NFS

Recommended for advanced users. NFS can provide higher transfer speeds. The NFS client feature is available only in specific Windows editions that include Client for NFS (Windows Server and Pro versions).
Doporučeno pro pokročilé uživatele. NFS může nabídnout vyšší přenosové rychlosti. Funkce klienta NFS je dostupná jen v některých edicích Windows, které obsahují Client for NFS (Windows Server a Pro verze).

1. Enable the NFS client

1. Zapněte klienta NFS

Download the Windows NFS client enable script: enable-nfs-client.ps1
Stáhněte skript pro zapnutí klienta NFS ve Windows: enable-nfs-client.ps1
Press the Start button, type powershell, right-click the first result, and select Run as administrator.
Stiskněte tlačítko Start, napište powershell, klikněte pravým tlačítkem na první výsledek a zvolte Spustit jako správce.
Use cd to navigate to the script location, then run .\enable-nfs-client.ps1. When it finishes, press Enter to restart your computer.
Pomocí příkazu cd přejděte do složky se skriptem a spusťte .\enable-nfs-client.ps1. Po dokončení stiskněte Enter a restartujte počítač.

2. Mount the drives

2. Připojte jednotky

Open a normal CMD session (do not run it as administrator).
Otevřete běžný CMD příkazový řádek (nespouštějte ho jako správce).
Run: mount SERVER_IP_OR_NAME:/DIRECTORY WINDOWS_LETTER:
Example: mount 192.168.1.230:/srv/data Z:.
Spusťte příkaz: mount SERVER_IP_NEBO_JMÉNO:/ADRESÁŘ PÍSMENO_VE_WINDOWS:
Příklad: mount 192.168.1.230:/srv/data Z:.
You can unmount an NFS drive with: umount LETTER:.
Jednotku NFS můžete odpojit příkazem: umount PÍSMENO:.
